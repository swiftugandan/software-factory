# Drop your inputs here

Put the Business Design Document and any supporting material in this folder before running
`/factory`:

- the BDD itself (any format Claude Code can read: md, txt, pdf, docx)
- data dictionaries, entity lists, field definitions
- wireframes or screenshots
- brand / tone notes, copy decks
- existing API contracts or integration specs

The factory reads everything here. A missing but referenced input is not a blocker — it
becomes a logged assumption. The more you provide, the fewer one-way assumptions the run has
to make.
